SKELETAL ANIMATION

A demo for skeletal animation rendering and ik solver
Game Engine: SD Engine
###################################
How to Build: 
- Use Visual Studio 2019
- No extra settings needed.

How to Use:
- WASD/QE and Mouse to move camera
- IJKL/UO to move IK effector (red dot)
- R to slow down animation
- F/G to loop through highlight bone
- H to switch between IK on head/hand

Known Issues: None
