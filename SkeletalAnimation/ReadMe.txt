DOOMENSTEIN

A 3D FPS
Game Engine: SD Engine
###################################
How to Build: 
- Use Visual Studio 2019
- No extra settings needed.

How to Use:
- KILL ARENA mode: kill enemy and player to earn points (1 per kill)
- WAVE RACING mode: stay in goal and kill wave enemy and player to earn points(1 per second in goal, 0.2 per kill)
- Press SPACE or controller START to start
- Press ESC or controller BACK to exit
- Play: WASD or controller left joystick to move
- Play: mouse or controller right joystick to aim
- Play: 1234... / LEFT/RIGHT / MOUSE WHEEL to change weapon
- Play: LEFT MOUSE / LEFT TRIGGER to fire current weapon
- Debug: N to posses next actor

Known Issues: None
